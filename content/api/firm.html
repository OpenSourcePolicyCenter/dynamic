Firm

ogusa.firm¶

ogusa.firm.get_K(L, r, p, method)[source]¶

Generates vector of aggregate capital. Use with the open economy options.

\[K_{t} = \frac{K_{t}}{L_{t}} \times L_{t}\]
Inputs:

L (array_like): aggregate labor r (array_like): the real interest rate p (OG-USA Specifications object): model parameters method (str): adjusts calculation dimensions based on ‘SS’ or

‘TPI’

Returns

aggregate capital demand

Return type

K (array_like)

ogusa.firm.get_KLratio_from_r(r, p, method)[source]¶

This function solves for the capital-labor ratio given the interest rate r and parameters.

\[\frac{K}{L} = \left(\frac{(1-\gamma)^\frac{1}{\varepsilon}}{\left[\frac{r + \delta - \tau^{corp}\delta^\tau}{(1 - \tau^{corp})\gamma^\frac{1}{\varepsilon}Z}\right]^{\varepsilon-1} - \gamma^\frac{1}{\varepsilon}}\right)^\frac{\varepsilon}{\varepsilon-1}\]
Parameters

  • r (array_like) – the real interest rate

  • p (OG-USA Specifications object) – model parameters

  • method (str) – adjusts calculation dimensions based on ‘SS’ or ‘TPI’

Returns

the capital-labor ratio

Return type

KLratio (array_like)

ogusa.firm.get_K_from_Y(Y, r, p, method)[source]¶

Generates vector of aggregate capital. Use with the open economy options.

\[K_{t} = \frac{Y_{t}}{Y_{t}/K_{t}}\]
Parameters

  • Y (array_like) – aggregate output

  • r (array_like) – the real interest rate

  • p (OG-USA Specifications object) – model parameters

  • method (str) – adjusts calculation dimensions based on ‘SS’ or ‘TPI’

Returns

the real interest rate

Return type

r (array_like)

ogusa.firm.get_Y(K, L, p, method)[source]¶

Generates aggregate output (GDP) from aggregate capital stock, aggregate labor, and CES production function parameters.

\[Y_{t} = Z_{t}\left[\gamma^{\frac{1}{\varepsilon}}K_{t}^{\frac{\varepsilon - 1}{\varepsilon}} + (1 - \gamma)^{\frac{1}{\varepsilon}}L_{t}^{\frac{\varepsilon - 1}{\varepsilon}}\right]^{\frac{\varepsilon}{\varepsilon - 1}}\]
Parameters

  • K (array_like) – aggregate capital

  • L (array_like) – aggregate labor

  • p (OG-USA Specifications object) – model parameters

  • method (str) – adjusts calculation dimensions based on ‘SS’ or ‘TPI’

Returns

aggregate output

Return type

Y (array_like)

ogusa.firm.get_r(Y, K, p, method)[source]¶

This function computes the interest rate as a function of Y, K, and parameters using the firm’s first order condition for capital demand.

\[r_{t} = (1 - \tau^{corp})(Z_t)^\frac{\varepsilon-1}{\varepsilon}\left[\gamma\frac{Y_t}{K_t}\right]^\frac{1}{\varepsilon} - \delta + \tau^{corp}\delta^\tau\]
Parameters

  • Y (array_like) – aggregate output

  • K (array_like) – aggregate capital

  • p (OG-USA Specifications object) – model parameters

  • method (str) – adjusts calculation dimensions based on ‘SS’ or ‘TPI’

Returns

the real interest rate

Return type

r (array_like)

ogusa.firm.get_w(Y, L, p, method)[source]¶

This function computes the wage as a function of Y, L, and parameters using the firm’s first order condition for labor demand.

\[w_t = (Z_t)^\frac{\varepsilon-1}{\varepsilon}\left[(1-\gamma)\frac{\hat{Y}_t}{\hat{L}_t}\right]^\frac{1}{\varepsilon}\]
Parameters

  • Y (array_like) – aggregate output

  • L (array_like) – aggregate labor

  • p (OG-USA Specifications object) – model parameters

  • method (str) – adjusts calculation dimensions based on ‘SS’ or ‘TPI’

Returns

the real wage rate

Return type

w (array_like)

ogusa.firm.get_w_from_r(r, p, method)[source]¶

Solve for steady-state wage w or time path of wages w_t given interest rate.

\[w = (1-\gamma)^\frac{1}{\varepsilon}Z\left[(\gamma)^\frac{1}{\varepsilon}\left(\frac{(1-\gamma)^\frac{1}{\varepsilon}}{\left[\frac{r + \delta - \tau^{corp}\delta^\tau}{(1 - \tau^{corp})\gamma^\frac{1}{\varepsilon}Z}\right]^{\varepsilon-1} - \gamma^\frac{1}{\varepsilon}}\right) + (1-\gamma)^\frac{1}{\varepsilon}\right]^\frac{1}{\varepsilon-1}\]
Parameters

  • r (array_like) – the real interest rate

  • p (OG-USA Specifications object) – model parameters

  • method (str) – adjusts calculation dimensions based on ‘SS’ or ‘TPI’

Returns

the real wage rate

Return type

w (array_like)
